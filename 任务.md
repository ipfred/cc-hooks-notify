

参考 Claude-Code-Notifier 里面发送钉钉和飞书消息的功能代码
参考 claude-code-hooks 的项目架构 


最终配置到claudecode settings.json 里的配置是固定的 通过调整项目配置 发送消息的样式渠道

```json
// .claude/settings.json （项目级，主文件保持简洁）
{
  "hooks": {
    "Notification": [
      {
        "matcher": "idle_prompt|permission_prompt",
        "hooks": [{ "type": "command", "command": "你的通知python脚本" }]
      }
    ],
    "Stop": [
      {
        "hooks": [{ "type": "command", "command": "你的完成python脚本" }]
      }
    ]
  }
}
```

3. 代码简洁，能很好的配置使用
